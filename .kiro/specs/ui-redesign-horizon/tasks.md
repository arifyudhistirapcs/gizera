# Implementation Plan: UI Redesign Horizon

## Overview

Redesign total UI aplikasi PWA ERP SPPG mengikuti style HR Attendee app. Implementasi dilakukan secara incremental: design system → core components → layout → redesign halaman existing → modul baru → routing & guards. Semua kode menggunakan Vue 3 + Vant UI 4 + Pinia + JavaScript.

## Notes

- Tasks marked with `*` are optional and can be skipped for faster MVP
- Each task references specific requirements for traceability
- Checkpoints ensure incremental validation
- Property tests use fast-check library and validate universal correctness properties
- All existing business logic (auth, attendance Wi-Fi/GPS, tasks offline sync, e-POD) is preserved — only UI layer changes
- New modules (Dashboard, Monitoring, Menu Planning, School Monitoring) are online-only since they are read-only/view-only
