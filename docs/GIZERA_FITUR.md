# Daftar Fitur Gizera

Gizera adalah platform manajemen operasional dapur untuk program Makan Bergizi Gratis (MBG). Tersedia dalam dua versi: **Gizera Web** (untuk pengelola dan manajemen) dan **Gizera Mobile** (untuk petugas lapangan).

---

## Gizera Web

Diakses melalui browser di laptop atau komputer. Digunakan oleh manajemen, staf dapur, dan administrator.

### Dashboard & Monitoring

| Fitur | Deskripsi |
|-------|-----------|
| Dashboard Kepala SPPG | Ringkasan harian operasional dapur: produksi, pengiriman, keuangan, dan stok dalam satu layar |
| Dashboard Kepala Yayasan | Pantau performa semua dapur (SPPG) di bawah yayasan secara agregat |
| Dashboard Admin BGN | Monitoring nasional seluruh yayasan dan SPPG, lengkap dengan peta sebaran sekolah dan supplier |
| Peta Sebaran | Peta interaktif yang menampilkan lokasi yayasan, dapur, sekolah, dan supplier dengan filter per kategori |
| Activity Tracker | Lacak seluruh aktivitas operasional harian dari persiapan hingga pengiriman selesai |
| Monitoring Pengiriman | Pantau status pengiriman makanan ke sekolah secara langsung |

### Perencanaan Menu

| Fitur | Deskripsi |
|-------|-----------|
| Perencanaan Menu Mingguan | Susun jadwal menu untuk satu minggu ke depan, lengkap dengan alokasi porsi per sekolah |
| Alokasi Porsi per Sekolah | Tentukan jumlah porsi kecil (kelas 1-3) dan besar (kelas 4-6) untuk setiap sekolah |
| Validasi Gizi Otomatis | Sistem otomatis memeriksa apakah menu memenuhi standar gizi minimum (600 kkal, 15g protein) |
| Persetujuan Menu | Alur persetujuan menu dari draft hingga disetujui oleh Kepala SPPG |
| Manajemen Resep | Kelola daftar resep lengkap dengan bahan, cara masak, informasi gizi, dan riwayat perubahan |
| Manajemen Komponen | Kelola bahan setengah jadi (nasi, lauk, sambal, dll) beserta resep dan stoknya |

### Supply Chain & Inventori

| Fitur | Deskripsi |
|-------|-----------|
| Manajemen Supplier | Daftar dan kelola data supplier lengkap dengan rating performa dan ketepatan pengiriman |
| Purchase Order (PO) | Buat dan kelola pesanan pembelian bahan baku dengan alur persetujuan |
| Penerimaan Barang (GRN) | Catat penerimaan barang dari supplier, cocokkan dengan PO, dan otomatis update stok |
| Manajemen Bahan Baku | Pantau stok bahan baku, terima notifikasi saat stok menipis |
| Stok Opname | Lakukan penghitungan fisik stok secara berkala dengan alur persetujuan dan export laporan |

### Logistik & Pengiriman

| Fitur | Deskripsi |
|-------|-----------|
| Data Sekolah | Kelola data sekolah penerima makanan: alamat, jumlah siswa, kontak, dan lokasi GPS |
| Tugas Pengiriman | Atur jadwal dan rute pengiriman makanan ke sekolah-sekolah |
| Tracking Ompreng | Lacak wadah makanan (ompreng) dari pengiriman hingga pengembalian dan pencucian |
| Ulasan Sekolah | Lihat ulasan dan rating dari sekolah tentang kualitas makanan dan layanan pengiriman |

### SDM & Kehadiran

| Fitur | Deskripsi |
|-------|-----------|
| Data Karyawan | Kelola data seluruh karyawan dapur: posisi, tanggal bergabung, dan status aktif |
| Laporan Absensi | Lihat rekap kehadiran karyawan harian dan bulanan, export ke Excel atau PDF |
| Konfigurasi Absensi | Atur metode absensi: validasi WiFi kantor atau GPS lokasi dapur |

### Keuangan

| Fitur | Deskripsi |
|-------|-----------|
| Aset Dapur | Catat dan kelola aset dapur (kompor, oven, freezer, dll) lengkap dengan jadwal perawatan |
| Arus Kas | Catat pemasukan dan pengeluaran harian dapur |
| Laporan Keuangan | Lihat ringkasan keuangan: anggaran vs realisasi, tren pengeluaran, dan export laporan |

### Audit & Kepatuhan SOP

| Fitur | Deskripsi |
|-------|-----------|
| Risk Assessment | Audit kepatuhan SOP dapur dengan formulir penilaian per kategori (higienitas, persiapan makanan, penyimpanan, dll) |
| Skor Risiko Otomatis | Sistem menghitung skor risiko berdasarkan penilaian checklist dan data operasional dapur |
| Riwayat Audit | Lihat riwayat semua audit yang pernah dilakukan, lengkap dengan skor dan tingkat risiko |

### Organisasi & Keamanan

| Fitur | Deskripsi |
|-------|-----------|
| Manajemen Yayasan | Kelola data yayasan dan SPPG di bawahnya (khusus Admin BGN) |
| Manajemen Pengguna | Buat dan kelola akun pengguna dengan 12+ peran berbeda sesuai tanggung jawab |
| Audit Trail | Rekam jejak seluruh aktivitas pengguna di sistem untuk keperluan audit |
| Konfigurasi Sistem | Atur parameter sistem seperti nama organisasi, batas stok minimum, dan radius pengiriman |

## Gizera Mobile

Diakses melalui browser di handphone. Digunakan oleh petugas lapangan, driver, dan manajemen saat di luar kantor.

### Untuk Driver & Asisten Lapangan

| Fitur | Deskripsi |
|-------|-----------|
| Daftar Tugas Harian | Lihat semua tugas pengiriman dan pengambilan ompreng hari ini |
| Detail Pengiriman | Lihat detail setiap pengiriman: sekolah tujuan, jumlah porsi, menu, dan rute |
| e-POD (Bukti Pengiriman Digital) | Catat bukti pengiriman dengan foto, tanda tangan digital penerima, dan koordinat GPS |
| Pengambilan Ompreng | Catat pengambilan wadah makanan dari sekolah setelah jam makan |
| Ulasan Pengiriman | Isi formulir ulasan kualitas makanan dan layanan dari pihak sekolah |

### Untuk Kepala SPPG

| Fitur | Deskripsi |
|-------|-----------|
| Dashboard Operasional | Ringkasan performa dapur hari ini: produksi, pengiriman, dan keuangan |
| Monitoring Pengiriman | Pantau status pengiriman semua driver secara langsung |
| Perencanaan Menu | Lihat dan kelola jadwal menu mingguan dari handphone |

### Untuk Kepala Yayasan

| Fitur | Deskripsi |
|-------|-----------|
| Dashboard Yayasan | Pantau performa semua dapur di bawah yayasan dari handphone |
| Audit Kepatuhan SOP | Isi formulir audit saat kunjungan lapangan ke dapur, lengkap dengan skor per kategori |
| Riwayat Audit | Lihat semua hasil audit sebelumnya dan lanjutkan draft yang belum selesai |

### Untuk Sekolah

| Fitur | Deskripsi |
|-------|-----------|
| Monitoring Sekolah | Pantau jadwal pengiriman makanan dan status pengiriman ke sekolah |

### Untuk Semua Pengguna

| Fitur | Deskripsi |
|-------|-----------|
| Absensi Digital | Check-in dan check-out kehadiran melalui handphone dengan validasi WiFi atau GPS |
| Profil Pengguna | Lihat dan kelola informasi akun pribadi |
| Mode Offline | Formulir audit dan data tugas tersimpan otomatis saat tidak ada koneksi internet, dan disinkronkan saat online kembali |

## Peran Pengguna

| Peran | Akses |
|-------|-------|
| Admin BGN | Dashboard nasional, manajemen yayasan & SPPG, manajemen pengguna |
| Kepala Yayasan | Dashboard yayasan, audit kepatuhan SOP, monitoring lintas dapur |
| Kepala SPPG | Seluruh fitur operasional dapur: menu, stok, logistik, SDM, keuangan |
| Ahli Gizi | Perencanaan menu, manajemen resep dan komponen |
| Akuntan | SDM, keuangan, inventori |
| Pengadaan | Supply chain: supplier, PO, penerimaan barang, inventori |
| Chef | Manajemen komponen dan resep |
| Driver | Tugas pengiriman, e-POD, pengambilan ompreng |
| Asisten Lapangan | Tugas pengiriman dan pengambilan ompreng |
| Sekolah | Monitoring pengiriman ke sekolah |
